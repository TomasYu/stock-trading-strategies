# 20周线趋势跟踪策略（30万资金版）

> 回测区间：2020-2025 | 总收益：27.69% | 最大回撤：-3.15% | 盈亏比：4.66

---

## 一、策略概述

放量突破20周线时买入，止损设在突破前低点，不设主动止盈，跌破20周线离场。30万资金版本在4万版基础上升级了持仓分散、金字塔加仓、板块控制和动态风险管理。

## 五、买入条件

以下**4个条件全部满足**时买入：

| 序号 | 条件 | 说明 |
|------|------|------|
| 1 | 20周线走平或拐头向上 | 斜率 ≥ 0 |
| 2 | 本周收盘站上20周线 | 周五收盘确认 |
| 3 | 上周收盘在20周线下方或附近 | 确认是"突破"行为 |
| 4 | 本周放量 | 成交量 > 5周均量 × 1.5 |

额外限制：
- 止损幅度不超过20%
- 该板块持仓未达2只上限
- 总持仓未达5只上限

**多只同时满足时，按近10周涨幅排序，优先买最强的。**

## 六、建仓规则

### 止损价

```
止损价 = 过去8周最低价 × 0.98
```

### 仓位计算

```
风险预算 = 总资金 × 风险比例（3%或4%，见动态风险）
买入股数 = 风险预算 ÷ (买入价 × 止损幅度)
```

举例（本金30万，风险比例3%）：
```
风险预算 = 300,000 × 3% = 9,000 元
某ETF买入价 0.54，止损幅度 10.8%
买入股数 = 9,000 ÷ (0.54 × 10.8%) ≈ 154,320 → 取154,300股
买入金额 = 154,300 × 0.54 = 83,322 元（占27.8%）
```

### 仓位上限

| 限制 | 规则 |
|------|------|
| 单只上限 | 总资金的 **30%** |
| 同时持有 | 最多 **5只** |
| 总仓位上限 | **90%** |
| 现金底线 | 至少保留 **10%** |
| 同板块上限 | 最多 **2只** |

## 七、金字塔加仓（最多2次）

| 次数 | 触发条件 | 加仓量 | 说明 |
|------|----------|--------|------|
| 第1次 | 盈利 ≥ **10%** | 正常计算 | 加仓时股价距20周线 ≤ 6% |
| 第2次 | 盈利 ≥ **20%** | 正常计算的 **50%** | 金字塔递减，越高加越少 |

加仓后单只总仓位不超过30%上限。

**只在盈利时加仓，绝不摊低亏损仓位。**

## 八、卖出规则

**不设主动止盈。让利润奔跑，直到趋势结束。**

| 优先级 | 信号 | 操作 |
|--------|------|------|
| 1 | 周五收盘价 < 止损价（前8周低点 × 0.98） | 下周一清仓 |
| 2 | 周五收盘价 < 20周均线 | 下周一清仓 |
| 3 | 持有超过12周，涨幅不到5% | 清仓换股 |

## 十、操作纪律

### 每周操作流程

```
每周五收盘后（约15:15）：

1. 检查上证指数是否在20周线上方
   → 不在：已有持仓按规则管理，不开新仓
   → 在：继续

2. 检查已有持仓（逐只检查）
   → 是否跌破止损价？→ 下周一清仓
   → 是否跌破20周线？→ 下周一清仓
   → 持有超过12周涨幅<5%？→ 下周一清仓换股
   → 盈利超10%/20%且未用完加仓次数？→ 考虑加仓
   → 以上都没有 → 继续持有

3. 检查30只ETF有无新的突破信号
   → 有：检查板块是否已满、持仓是否已满
   → 算仓位和止损价，下周一买入

4. 检查空仓资金是否买了货币基金
   → 买入前先确认下周有没有要建仓的

5. 记录交易日志
```

### 铁律

1. **不盘中决策**：周五收盘后决策，周一执行
2. **不摊低成本**：亏损仓位永远不加仓
3. **不扛亏损**：触发止损必须走
4. **不主动止盈**：让利润奔跑
5. **不追涨停**：买不进去就放弃
6. **不借钱炒股**：30万亏完就亏完
7. **不板块集中**：同板块最多2只
