# 20周线趋势跟踪策略（4万资金·最终版）

> 回测区间：2020-2025 | 总收益：23.27% | 最大回撤：-6.12% | 胜率：62.5% | 盈亏比：3.47

## 一、策略概述

放量突破20周线时买入，止损设在突破前的低点，不设主动止盈，完全靠跌破20周线离场。

核心逻辑：**突破确认趋势启动 → 前低止损给足空间 → 让利润奔跑直到趋势结束**

## 四、买入条件

以下**4个条件全部满足**时，产生买入信号：

| 序号 | 条件 | 说明 |
|------|------|------|
| 1 | 20周线走平或拐头向上 | 20周线值 ≥ 上周20周线值 |
| 2 | 本周收盘站上20周线 | 周五收盘价 > 20周均线值 |
| 3 | 上周收盘在20周线下方或附近 | 上周收盘价 ≤ 20周均线值 × 1.02，确认是"突破"行为 |
| 4 | 本周放量 | 本周成交量 > 近5周平均成交量 × 1.5 |

**多只同时满足时，选近10周涨幅最大的（动量最强）。**

## 五、建仓规则

### 买入时机

周五收盘确认满足全部4个条件 → **下周一开盘买入**

### 止损价确定

```
止损价 = 过去8周最低价 × 0.98（留2%缓冲）
```

举例：
```
某ETF本周放量突破20周线，收盘价 3.84 元
过去8周最低价 = 3.60 元
止损价 = 3.60 × 0.98 = 3.53 元
止损幅度 = (3.84 - 3.53) / 3.84 = 8.1%
```

如果止损幅度超过20%，不做，风险不可控。

### 仓位计算

```
最大亏损金额 = 总资金 × 3%
买入股数 = 最大亏损金额 ÷ (买入价 × 止损幅度)
```

举例（本金4万）：
```
最大亏损金额 = 40000 × 3% = 1200 元
买入股数 = 1200 ÷ (3.84 × 8.1%) = 1200 ÷ 0.31 ≈ 3800 股
买入金额 = 3800 × 3.84 = 14592 元（占36.5%）
```

### 仓位上限

| 限制 | 规则 |
|------|------|
| 单只上限 | 总资金的 **60%** |
| 同时持有 | 最多 **2只** |
| 现金底线 | 至少保留总资金的 **10%** |
| 实际仓位 | 取"风险计算仓位"和"仓位上限"中较小的 |

### 操作步骤

1. 周五收盘后检查各ETF是否满足4个买入条件
2. 确定买入标的
3. 查看该ETF过去8周最低价，算出止损价
4. 用公式算出买入股数
5. **把止损价写下来贴在电脑旁边**
6. 下周一开盘买入

---

## 七、卖出规则

**不设主动止盈。涨多少都拿着，直到趋势结束。**

按优先级从高到低，满足任一条件即执行：

| 优先级 | 信号 | 操作 |
|--------|------|------|
| 1 | 周五收盘价 < 止损价（前8周低点 × 0.98） | 下周一清仓 |
| 2 | 周五收盘价 < 20周均线 | 下周一清仓 |
| 3 | 持有超过12周，涨幅不到5% | 清仓换股 |

就这三条，简单明确。

**为什么不设止盈**：趋势跟踪的核心是"让利润奔跑"。只要20周线没破，趋势就还在，没理由下车。止盈只会让你过早离场，错过后面的大肉。

## 九、操作纪律

### 每周操作流程

```
每周五收盘后（约15:15）：

1. 检查上证指数是否在20周线上方
   → 不在：已有持仓按规则管理，不开新仓
   → 在：继续

2. 检查已有持仓
   → 是否跌破止损价？→ 下周一清仓
   → 是否跌破20周线？→ 下周一清仓
   → 持有超过12周涨幅<5%？→ 下周一清仓换股
   → 以上都没有 → 继续持有
   → 盈利超过10%且未加仓？→ 考虑加仓

3. 检查ETF池有无新的突破信号
   → 有：算仓位和止损价，下周一买入
   → 没有：等待

4. 记录交易日志
```

### 时间安排

| 项目 | 时间 |
|------|------|
| 复盘 | 每周五收盘后，花15-20分钟 |
| 操作 | 周一开盘前下单 |
| 盯盘 | **不需要**，周中不看盘 |

### 铁律

1. **不盘中决策**：所有决策在周五收盘后做，周一机械执行
2. **不摊低成本**：亏损的仓位永远不加仓
3. **不扛亏损**：触发止损必须走，没有"再看看"
4. **不主动止盈**：涨多少都不卖，等跌破20周线再说
5. **不追涨停**：买不进去就放弃
6. **不借钱炒股**：4万亏完就亏完，绝不加杠杆
